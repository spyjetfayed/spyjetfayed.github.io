# copilot-chat

https://github.com/chep/copilot-chat.el

To append a section of code to a prompt:

  1. Select the region
  2. call copilot-chat-custom-prompt-selection
  3. enter the prompt in the mini buffer
  4. press enter
  5. Go to the copilot buffer. The input should be there.
  6. Inside the “chat-input” section type , and select copilot-chat-prompt-send

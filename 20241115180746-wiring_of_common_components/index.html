# Pots

  1. Center pin is output
  2. One pin to VCC
  3. The other pin to ground

Switch ground and VCC to change how you want the output to respond in relation to the direction you turn the knob.

# 3 Pin

  1. One pin to VCC
  2. One pin to ground
  3. Center to digital input

When in ON position, it will connect VCC to input pin. When in OFF position it will connect input pin to ground.
